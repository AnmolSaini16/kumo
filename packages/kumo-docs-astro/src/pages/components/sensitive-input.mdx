## With Copy Button

<p>
  Pass <code class="rounded bg-kumo-control px-1 py-0.5 text-sm">onCopy</code> to
  enable a copy-to-clipboard button. An anchored toast confirms the action.
</p>

```tsx
<SensitiveInput
  label="API Key"
  defaultValue="sk_live_abc123xyz789"
  onCopy={() => {}}
/>
```

## Internationalization

<p>
  Pass a <code class="rounded bg-kumo-control px-1 py-0.5 text-sm">labels</code> prop
  to override the default English strings for tooltips, aria labels, and the
  copy confirmation toast.
</p>

```tsx
<SensitiveInput
  label="Clé API"
  defaultValue="sk_live_abc123xyz789"
  onCopy={() => {}}
  labels={{
    copyAction: "Copier dans le presse-papiers",
    copyTooltip: "Copier",
    copiedToast: "Copié !",
    reveal: "Afficher la valeur",
    revealTooltip: "Afficher",
    hide: "Masquer la valeur",
    hideTooltip: "Masquer",
  }}
/>
```
